Personal Health Record Tool - Terms of Use & Privacy Notice

Last updated: July 18, 2026

This is a personal, single-user software tool used by its owner to download a copy of their own medical records from their healthcare provider's patient-facing API and view them locally. It is not a commercial product and is not offered to, or used by, any other people or organizations.

What the tool does

  • Connects to a healthcare provider's patient FHIR API only after the owner logs in with their own patient-portal credentials and grants consent.
  • Reads a copy of the owner's own health information (read-only). It never writes to, changes, or deletes anything in the provider's records.

How data is stored and used

  • Data stays on the owner's own device. Retrieved records are saved as local files on the computer running the tool.
  • The tool does not transmit, upload, sell, share, or disclose the owner's health data to the developer or any third party.
  • Data is used solely to display the owner's records back to the owner.
  • No advertising, no analytics, no tracking.

Access and deletion

  • The owner can obtain a complete copy of all data the tool has stored (the local files themselves).
  • The owner can delete all stored data at any time by deleting those local files.
  • Access to the provider's API can be revoked at any time from within the provider's patient portal ("Linked Apps and Devices").

Not a HIPAA-covered entity

This tool is operated by an individual for personal use and is not a HIPAA-covered entity or business associate. Because the owner is the only person who uses it and the only person whose data it accesses, health information does not leave the owner's control.

Contact

This is a personal-use application. Questions about the owner's health records should be directed to the owner's healthcare provider.
